
# Food Sales App

This project covers the basic food ordering process from selecting categories and items until buying the shopping cart.

## Used tools
Considering the goal of the project is to generate the app using basic technologies only. It implements the following tools:
* HTML
* Tailwind CSS
* JS

## Design
* Event handlers
* Event listeners
* Event delegation

## Storage
* Local Storage

## Usage instructions

When the app is opened, the user will land on the food categories page. The user selects one category, and it will open all the items that can be chosen. In order to buy an item, the user should select the size of the item and increment the quantity to a number bigger than 0. Once the item is selected, the user can go to the shopping cart icon and will be able to buy it. The success message will be displayed after the sale is completed successfully.
